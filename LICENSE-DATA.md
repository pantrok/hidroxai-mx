# Licencia de los datos

El **código** de este repositorio se distribuye bajo licencia **MIT** (ver `LICENSE`).

Los **datos derivados** producidos por este proyecto (el dataset curado HidroXAI-MX,
depositado en Zenodo con DOI) se publican bajo **Creative Commons Atribución 4.0
Internacional (CC BY 4.0)**, compatible con los Términos de Libre Uso MX de las fuentes.

> Los datos NO se versionan en GitHub: viven en el remoto DVC (Cloudflare R2) y en la
> versión publicada de Zenodo. Este repositorio solo contiene código y punteros `.dvc`.

## Atribución obligatoria de las fuentes
- **CONAGUA — Sistema de Información Hidrológica (SIH) y BANDAS**: Términos de Libre Uso MX.
- **SMN (CONAGUA)**: Términos de Libre Uso MX.
- **INEGI** (CEM, red hidrográfica): Términos de Libre Uso del INEGI.
- **CICESE-CLICOM**: uso académico con cita obligatoria —
  "Datos climáticos diarios del CLICOM del SMN a través de su plataforma web del CICESE
  (http://clicom-mex.cicese.mx)".
- **CONABIO** (espejo hidrográfico): CC BY-NC 2.5 MX.
